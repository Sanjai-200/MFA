Configuration and architecture

Risk Model
model.pkl (device, location, loginCount, hour, failedAttempts)
Prediction Logic
0 = Safe → Login directly | 1 = Risky → OTP email sent
OTP Validity
60 seconds — sent via Gmail SMTP
Auth Method
JWT Token — 8 hour expiry
Database
SQLite (change DB_PATH in app.py)
Location Detection
ipify.org → ipapi.co → ipwho.is (3-API fallback)
Super Admin
Hardcoded in app.py — cannot be deleted or blocked
Default New Role
user (assigned on signup)
